Notas antes de começar o exercício:

  1. Inicialmente, entre no GitHub e crie um usuário para você.
  2. Após isso, crie um repositório com o nome "Programação_1"
  3. Crie uma pasta com o nome "Exercícios_de_sala"
    Todos os exercícios que você fizer em sala deverão ser armazenados nessa pasta.
    Após testar o seu código e ver se funciona corretamente, crie dois arquivos conforme esse modelo:
    * Arquivo 1: atividade-2022-05-27.c
    * Arquivo 2: atividade-2022-05-27.txt
    -> O "Arquivo 1" deverá conter o seu código fonte.
    -> O "Arquivo 2" deverá conter os testes que você fizer na execução do seu programa.
  4. Envie um e-mail para o professor informando:
    • seu nome,
    • sua matrícula,
    • sua turma,
    • seu endereço no GitHub.
